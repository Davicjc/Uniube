Apaga toda a coleção exercises do Firestore e recria os exercícios com dados de animação reais (animFrames + motionRange).

⚠ Atenção: O botão vermelho tenta apagar todos os exercícios antes de criar os novos. Se aparecer "Missing or insufficient permissions", use o botão verde (só adiciona, não apaga) ou siga as instruções abaixo.
Se der erro de permissão ao deletar:
1. Acesse console.firebase.google.com
2. Projeto faculdade-368a4 → Firestore → Rules
3. Substitua as regras por:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}

4. Publique as regras e clique novamente no botão vermelho.
5. Após o seed, restaure as regras originais.
